# Omnify
A plugin that "omnify"'s any midi source, allowing you to play any midi instrument (virtual or not) like it was an omnichord, strumming on the mod touch plate or modwheel of your keyboard and using pads to select chord quality and notes to select the root of chords.

Outputs separate midi channels for chords vs strums.
